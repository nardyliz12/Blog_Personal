Machine Learning

Clasificación de imágenes para la detección de Retinopatías Diabéticas con Machine Learning

Este proyecto consistió en desarrollar un sistema de visión computacional con Machine Learning para detectar retinopatías diabéticas en imágenes de retina. Se utilizó un dataset de Kaggle para entrenar modelos de clasificación basados en redes neuronales convolucionales (CNN), logrando distinguir entre imágenes sin signos de la enfermedad y aquellas con retinopatía diabética en distintos niveles. El resultado fue una herramienta que demuestra el potencial del Deep Learning para apoyar el diagnóstico médico y la detección temprana de complicaciones visuales que va de la mano con la medicina utilizando conocimientos de Machine Learning.

Python TensorFlow CNN Kaggle
IoT

AIRVIBE: Sistema Integrado de Monitoreo de Calidad del Aire

AIRVIBE es un proyecto enfocado en el desarrollo de un prototipo de dispositivo de monitoreo ambiental capaz de medir en tiempo real la calidad del aire, donde brinda datos de niveles de Monóxido de Carbono, Material Particulado PM10, PM2.5 y los niveles de ruido. Los datos recolectados son accesibles mediante una interfaz web y el sistema emite alertas ante niveles altos de contaminación. Este proyecto fue diseñado en el curso de Proyectos de Ingeniería I con el objetivo de brindar información clara a los ciudadanos y apoyar a las autoridades locales en la toma de decisiones para mejorar el entorno, contribuyendo así a los Objetivos de Desarrollo Sostenible número 13 que hacen énfasis a acción por el clima.

Arduino IoT Web ODS 13
Deep Learning

Implementación de un Sistema Multi-Agente de Ajedrez con Aprendizaje por Refuerzo Profundo

Este proyecto se centró en desarrollar un entorno de ajedrez y aplicar el algoritmo de Optimización de Políticas Proximales (PPO) para entrenar agentes inteligentes mediante aprendizaje por refuerzo profundo. Se implementaron dos enfoques: Double Agent, donde dos agentes con redes neuronales independientes competían entre sí para mejorar sus estrategias, y Single Agent, donde un único agente aprendía a jugar con ambos colores del tablero optimizando recursos computacionales. El resultado fue un sistema capaz de aprender y perfeccionar su desempeño en ajedrez de forma autónoma, combinando técnicas avanzadas de inteligencia artificial y aprendizaje profundo.

Python PyTorch PPO RL

TinyML

Sistema Inteligente para la Detección de Fugas de Agua con TinyML

Este proyecto tuvo como objetivo desarrollar un sistema inteligente para la detección de fugas de agua, buscando aportar soluciones a la problemática de la escasez hídrica en el Perú. El sistema fue diseñado en el curso de Fundamentos de Diseño utilizando Edge Impulse para entrenar y desplegar modelos de Machine Learning en hardware embebido, permitiendo identificar fugas de manera temprana y con ello, buscar optimizar el uso de los recursos hídricos, asimismo, contribuir al cumplimiento del Objetivo de Desarrollo Sostenible (ODS) 6: Agua limpia y saneamiento.

Despliegue de servicio de música por streaming con AWS

El proyecto realizado implementó un servicio de música por streaming utilizando la plataforma AWS (Amazon Web Services) y la aplicación Airsonic, un servidor multimedia de código abierto. El objetivo fue desplegar un servicio funcional que permitiera a los usuarios acceder, reproducir y gestionar música en línea a través de una red. Para ello, se configuró una instancia EC2 en AWS como servidor principal, donde se instaló y configuró Airsonic junto con los servicios necesarios (Docker). Esta práctica permitió aplicar conceptos de protocolos de red, transmisión de datos, direccionamiento IP y servicios en la nube, comprendiendo cómo se integran en una arquitectura moderna de streaming.

AWS EC2 Docker Networking
